准备工作

安装Git

  1. 访问 Git - Downloading Package 下载并安装Git。
  2. 安装时请确保勾选“Add to PATH”以便在命令行中直接使用Git命令。

安装Node.js

  1. 访问 Node.js 中文网 下载并安装Node.js。
  2. 安装完成后，打开命令行工具，输入 node -v 来验证Node.js是否安装成功。

配置GitHub仓库

  1. 登录GitHub，创建一个新的仓库，仓库名通常为 用户名.github.io。
  2. 在仓库的设置中，找到“SSH and GPG keys”部分，新建一个SSH key。

生成SSH Key

在本地计算机上生成SSH key，并将其添加到GitHub：

  • 按提示操作，默认路径即可。
  • 找到生成的 id_rsa.pub 文件（通常位于 C:\Users\用户名\.ssh\），用记事本打开并复制其内容。
  • 在GitHub的SSH key设置中，新建一个key，粘贴 id_rsa.pub 的内容，并保存。
